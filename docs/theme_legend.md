# Theme Legend

Use this legend to keep categorization consistent across runs.

| Theme | Definition | Example signal |
|---|---|---|
| App Performance | Crashes, lag, hangs, loading delays, runtime errors | "App freezes when market opens" |
| Trading Charges/Pricing | Brokerage, hidden charges, pricing confusion, fee dissatisfaction | "Charges are too high for small trades" |
| Customer Support | Support responsiveness, ticket handling quality, resolution loop | "Support keeps asking the same details" |
| Features Performance | Core feature behavior (watchlist, charts, portfolio, notifications, UX flow) | "Portfolio chart is not updating" |
| KYC/Statements/Withdrawals | KYC verification delays, statement availability, withdrawal/payout issues | "Withdrawal is pending without reason" |

Rules:
- Always map to one of the fixed 5 themes.
- If a review fits multiple themes, choose the dominant user pain.
- Keep theme naming exactly stable for weekly comparison.
